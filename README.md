# Handwritten Digit Recognition

A Machine Learning project that recognizes handwritten digits using the MNIST dataset and compares the performance of KNN, SVM, and MLP classifiers.

## Features
- Image preprocessing and normalization
- KNN, SVM, and MLP model implementation
- Accuracy comparison of models
- Random digit prediction visualization
- Confusion matrix generation

## Technologies Used
- Python
- NumPy
- Matplotlib
- TensorFlow/Keras
- Scikit-learn

## Results
The project evaluates and compares KNN, SVM, and MLP models using accuracy scores and confusion matrices to identify the best-performing classifier.
